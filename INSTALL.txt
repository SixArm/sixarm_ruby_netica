
= SixArm.com Ruby Gem Install


== Easy Way

  gem install sixarm_ruby_netica --source http://sixarm.com 


== Secure Way

First get our SixArm.com secure signature gem certificate,
then add us to your gem cert list and gem sources list.
When you do this once, it works for all our SixArm gems.

  wget http://sixarm.com/sixarm.pem
  gem cert --add sixarm.pem   
  gem sources --add http://sixarm.com

Install the gem with testing and high security.

  gem install sixarm_ruby_netica --test --trust-policy HighSecurity
